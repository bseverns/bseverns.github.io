# Legacy JavaScript Boundary

These files are retained for preserved archive pages under `2d/`, `3d/`, and `text/`.

Do not use this directory for modern Atlas, Studio, Teaching, Press, Contact, or homepage code. New progressive-enhancement work belongs in focused top-level scripts such as `/js/site.js`, `/js/atlas.js`, `/js/diagram-loader.js`, or `/js/status-hook.js`.
