<p>What this node now points to is less a static clip bin than a playable visual runtime. In the current rig, <code>nw_wrld</code> provides the sequenced endpoint and <code>wrld_on</code> carries workspace modules, assets, and mappings, so source material can be reshaped, reloaded, and performed as part of the larger scene system.</p>
