# Legacy JavaScript Boundary

The files below are not part of the modern Atlas, Studio, Teaching, Press, or Contact surfaces. They are retained because the older archive pages in `2d/`, `3d/`, and `text/` still reference them directly.

- `js/jquery.min.js`
- `js/jquery.easing.min.js`
- `js/modernizr.custom.42534.js`
- `js/jquery.waitforimages.js`
- `js/typed.js`
- `js/masonry.pkgd.min.js`
- `js/imagesloaded.pkgd.min.js`
- `js/jquery.jkit.1.2.16.min.js`
- `js/script.js`

Modern pages should continue to use `js/site.js`, `js/diagram-loader.js`, and focused progressive-enhancement scripts. Do not add the legacy stack to new pages unless an archive page is being preserved as-is.
