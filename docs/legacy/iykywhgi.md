# IYKYWHGI

*IYKYWHGI* is a 2019 feedback system connecting a camera, floor input, Processing, MIDI, and a VCV Rack sound patch. The archived program compares the camera image against a stored frame, maps the difference to MIDI, and uses serial input to switch parts of the loop on and off.

The installation photograph keeps that mechanism visible: webcam and monitor above a compact computer, exposed cables, a control enclosure, and two floor-level sensor boxes. Image, movement, control, and sound remain one unstable circuit rather than separate media.

<p class="origin-callout">One person-free installation photograph, source code, and the sound patch survive. The archive confirms the 2019 system but does not currently establish a venue, so none is claimed here.</p>
